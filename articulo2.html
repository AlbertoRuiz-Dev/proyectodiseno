Introducción a JavaScript

Conceptos básicos para principiantes en JS.

JavaScript es un lenguaje de programación esencial para añadir interactividad a las páginas web. Se ejecuta en el navegador y te permite manipular el DOM, responder a eventos y mucho más. Para empezar, incluye un script en tu HTML: 

<script>
    console.log("¡Hola, mundo!");
</script>

Algunos conceptos básicos incluyen variables, funciones y eventos. Por ejemplo, para declarar una variable y mostrar un mensaje: 

let nombre = "Alberto";
alert("Hola, " + nombre);

Puedes aprender más practicando con eventos como clics de botón o utilizando funciones para reutilizar código. ¡JavaScript es el primer paso hacia la programación dinámica en la web!
